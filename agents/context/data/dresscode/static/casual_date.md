# 캐주얼 데이트 (Casual Date) 드레스코드

캐주얼 데이트는 단정함과 자연스러움의 균형이 핵심이다. 깔끔한 셔츠·블라우스·니트 또는 무지 티셔츠 위에 가벼운 자켓이나 가디건을 걸치고, 하의는 슬랙스·치노·청바지 중에 자리 분위기에 맞춰 선택한다. 세미머슬핏 또는 와이드 핏 데님이 무난하고, 지나치게 트레이닝 룩으로 가지 않는 선에서 멋을 낸다.

신발은 깔끔한 스니커즈 또는 로퍼·더비 슈즈가 활용도가 높다. 색상은 흰색·베이지·회색·네이비·검정 같은 베이직 톤을 베이스로 잡고 포인트 컬러는 자유롭게 사용해도 좋지만, 위아래 모두 채도 높은 색은 피한다. 식당·카페·산책 등 일반 데이트 코스에서는 격식 중간, 갤러리·공연·호텔 다이닝처럼 분위기 있는 장소에서는 격식 상한 쪽으로 맞춘다.
