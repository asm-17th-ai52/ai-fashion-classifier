# 일반 (General) 드레스코드

특정 event_type에 해당하지 않거나 분류가 모호한 경우의 폴백(fallback) 가이드다. 비즈니스 캐주얼과 깔끔한 캐주얼 사이의 넓은 격식 구간을 커버한다. 상의는 셔츠·블라우스·니트·티셔츠 중에서 자유롭게 고르고, 하의는 슬랙스·치마·치노·청바지 모두 허용된다. 아우터는 블레이저·자켓·가디건으로 격식을 조절한다.

신발은 구두·로퍼·스니커즈 모두 가능하며 상하의 격식에 맞춘다. 색상은 흰색·회색·검정·네이비·베이지 같은 무채색·중성 톤을 기본으로 두고, 극단적 채도(빨강·노랑)는 한 점 포인트 이내로 제한한다. 자리의 종류가 더 구체적으로 파악되면 interview/business_meeting/wedding_guest/outdoor_activity 등 전용 event_type으로 전환하는 것이 좋고, 이 항목은 분류 신뢰도가 낮을 때만 사용한다.
