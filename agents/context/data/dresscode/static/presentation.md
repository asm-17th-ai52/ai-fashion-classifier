# 발표 (Presentation) 드레스코드

발표는 청중 앞에 서는 자리이므로 단정함과 신뢰감을 동시에 줘야 한다. 표준은 감청색·회색 계열 슈트에 흰색 또는 옅은 톤 셔츠·블라우스 조합이며, 여성은 무채색 원피스에 블레이저를 더한 세미 정장도 무난하다. 장시간 발표를 고려해 활동성이 있는 슬랙스·무릎 길이 치마가 권장된다.

상의 색은 카메라·프로젝터 빛에 잘 받는 흰색·회색·네이비가 안전하며, 검정만 단독으로 입으면 답답해 보일 수 있어 블라우스나 셔츠로 톤을 띄운다. 신발은 구두·로퍼를 신고 굽 높이는 오래 서 있을 수 있는 정도로 조절한다. 빨강·노랑·핑크·코랄·연두 같은 시선을 분산시키는 채도는 피한다. 학술 발표는 세미 정장 하한, 사내 임원 보고나 외부 제안 발표는 정장 상한에 맞춘다.
